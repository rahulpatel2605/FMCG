# ⚡ Aethera — Premium FMCG Dealer & Inventory Console

Aethera is a next-generation, high-performance, and visually stunning FMCG (Fast-Moving Consumer Goods) Dealer Portal. Designed with sleek modern design principles, dark-mode glassmorphism, responsive dashboard charts, dynamic trade schemes, and real-time inventory telemetry.

---

## ✨ Features

- **Live Executive Telemetry Dashboard**: Get instant insight into stock valuation, overall inventory health percentage, active regional trade schemes, and critical stockout risk level alerts.
- **Dynamic Chart Insights**: Integrates interactive area charts for secondary logistics flows and elegant custom bar graphs representing category weight allocations.
- **Proactive Inventory & Restock Controller**: Interactive grid list displaying brand SKU statuses, stock counts, batch codes, and a fast-dispatch cart queuing system.
- **Algorithmic Trade Scheme Engine**: Automated trade discount calculations, margin boosters (e.g. Colgate's Buy 5 get 1 free), cash back values, and free-goods dispatch logistics in real-time.
- **Premium Order Dispatch System**: Interactive ledger sheet presenting trade margins, GST calculations, net totals, and secondary logistics carrier assignments.
- **Stand-alone Browser Preview**: A fully integrated `interactive_preview.html` to instantly run, demonstrate, and test the entire premium experience straight from the browser without any toolchains.

---

## 🛠️ Technology Stack

- **Framework**: [React](https://react.dev/) + [Vite](https://vite.dev/) (Lightning fast development & hot modular reloads)
- **Styling**: Modern Vanilla CSS, Custom HSL Hues, Glassmorphism, and Fluid Transitions.
- **Icons**: [Lucide React](https://lucide.dev/) (Clean and crisp vector iconography)
- **Visualization**: [Recharts](https://recharts.org/) (Dynamic SVG-based charting components)
- **Fonts**: Inter & Outfit (Premium Google Fonts)

## 📦 Getting Started

To run the application locally, you'll need Node.js and a package manager installed.

### 1. Installation

Clone the repository and install the dependencies:

```bash
git clone https://github.com/rahulpatel2605/FMCG.git
cd FMCG
npm install
```

### 2. Running in Development Mode

Launch the local development server:

```bash
npm run dev
```

The application will launch on your default browser at `http://localhost:3000`.

### 3. Production Build

To compile a highly optimized static production package:

```bash
npm run build
```

The build output will be exported to the `dist` directory.

## 🖥️ Standalone Instant Preview
